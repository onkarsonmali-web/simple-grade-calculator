# 🎯 Student Grade Calculator

## 📌 Project Overview
This project is a Python-based Student Grade Calculator that takes student marks as input and assigns grades based on predefined criteria. It also provides encouraging messages for each grade.

## 🎯 Objectives
- To understand decision-making using if-elif-else
- To practice loops and input validation
- To implement functions in Python
- To handle errors using exception handling

## ⚙️ Features
- Grade calculation (A, B, C, D, F)
- Input validation (marks must be between 0–100)
- Error handling for invalid inputs
- Encouraging messages for students
- User-friendly output
